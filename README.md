# Ethereum core 연동 project

---

## About

This is a starter boilerplate app I've put together using the following technologies:

* [Express](http://expressjs.com)
* [Web3](https://web3js.readthedocs.io/en/v1.2.4/)

## Installation

```bash
npm install
```

## Running Server

```bash
node ethServer.js
```
